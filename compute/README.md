# Compute

The compute layer is responsible for hosting and scaling application workloads within the AWS platform.

This section demonstrates how Amazon EC2, Auto Scaling Groups, and Application Load Balancers work together to provide a highly available, fault-tolerant, and scalable application architecture.

The design follows AWS Well-Architected Framework recommendations by separating compute resources from networking, database, and monitoring components.

---

## Components

- Amazon EC2
- Auto Scaling Group (ASG)
- Application Load Balancer (ALB)
- Security Groups

---

## Design Principles

- High Availability
- Elastic Scaling
- Fault Tolerance
- Security
- Operational Simplicity
