# Serverless

Serverless computing enables applications and operational tasks to run without managing underlying servers. AWS Lambda executes code in response to events and automatically scales based on demand.

This section demonstrates how serverless automation can simplify cloud operations by performing routine maintenance tasks, integrating with monitoring services, and reducing manual effort.

---

## Components

- AWS Lambda
- Amazon EventBridge
- Amazon CloudWatch
- Amazon SNS
- AWS Identity and Access Management (IAM)

---

## Design Principles

- Event-Driven Architecture
- Operational Automation
- Scalability
- Cost Efficiency
- Least Privilege Access
