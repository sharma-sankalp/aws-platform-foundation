# Database

The database layer stores application data securely while ensuring high availability, durability, and scalability.

This section demonstrates how Amazon Relational Database Service (Amazon RDS) integrates with application workloads deployed on Amazon EC2 within a private network.

The database is isolated from the public internet and accessed only by trusted application servers through Security Groups.

- Amazon RDS
- DB Subnet Group
- Private Subnets
- Security Groups
- Automated Backups
