# Monitoring

Monitoring is a critical part of operating production workloads on AWS.

This section demonstrates how monitoring, alerting, logging, and event-driven notifications improve operational visibility and reduce incident response time.

The monitoring architecture combines Amazon CloudWatch, Amazon EventBridge, AWS Lambda, and Amazon SNS to automate operational tasks and notify engineers when important events occur.

---

## Components

- Amazon CloudWatch
- Amazon EventBridge
- Amazon SNS
- AWS Lambda
