# Networking

Networking is the foundation of every AWS platform.

This section demonstrates how networking components work together to provide secure communication between internet-facing services, private workloads, and backend resources.

Topics covered include:

- Amazon VPC
- Public and Private Subnets
- Route Tables
- Internet Gateway
- NAT Gateway
- Security Groups
- Network Design
- VPC Peering

Each document explains not only *how* to configure the service, but also *why* the architecture is designed that way.
